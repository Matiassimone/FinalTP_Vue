# VUE Final TP

this app use [VueJS](https://vuejs.org), [Vuex](https://vuex.vuejs.org/), [VueRouter](https://router.vuejs.org/) and [Vuetify](https://vuetifyjs.com) 


## Steps to Start

### - For run the server
 - Open Terminal
 - Go the server folder
 - Type npm install
 - Type node index
### - For run the web
 - Open Terminal
 - Go the server folder
 - Type npm install
 - Type npm run serve
